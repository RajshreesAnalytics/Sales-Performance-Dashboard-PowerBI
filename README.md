# Sales Performance Dashboard (Power BI)

This project presents an interactive Power BI dashboard analyzing sales performance across multiple dimensions including time, category, region, and product.

## Key Features
- KPI metrics: Total Revenue, Total Orders, Total Quantity
- Custom metric: Average Revenue per Order
- Monthly revenue trend analysis
- Category and region-wise sales comparison
- Top product analysis
- Interactive slicers for Category, Region, and Order Date
- Insight cards for Top Product and Top Region

## Tools Used
- Power BI
- DAX
- Power Query
- Excel dataset

## Dashboard Insights
This dashboard helps answer questions such as:
- How does revenue change over time?
- Which category generates the highest revenue?
- Which region performs best?
- Which product contributes the most revenue?
- What is the average revenue per order?

## Files Included
- `.pbix` Power BI dashboard file
- Source dataset
- Final dashboard screenshot

## Project Objective
The goal of this project is to build a clean, interactive, and business-focused sales dashboard using Power BI and present insights in a portfolio-ready format.
